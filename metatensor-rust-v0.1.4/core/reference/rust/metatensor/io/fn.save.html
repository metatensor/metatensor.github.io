pub fn save(path: impl AsRef<Path>, tensor: &TensorMap) -> Result<(), Error>
Expand description

Save the given tensor to a file.

If the file already exists, it is overwritten.

The format used is documented in the load function, and is based on numpy’s NPZ format (i.e. zip archive containing NPY files).
